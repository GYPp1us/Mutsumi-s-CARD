# 银行卡比例与绘图手势设计

## 目标

- 所有新卡片、绘图画布和缩略图采用 ISO ID-1 竖向比例：`53.98 / 85.60 = 0.6306075`。
- 学习页卡片宽度最多占可用宽度 `80%`，左右各至少保留 `10%`。
- 学习页和竖屏录入首页在安全区内保留 `16dp` 内容边距。
- 旋转设备不清空 key、底图、笔迹、工具、步骤和相机。
- 移动缩放在整个手势期间连续跟手，缩放锚点为每帧双指实时质心。

## 状态与坐标

绘图文档由 `DrawingViewModel` 持有，Composable 仅持有与具体像素视口相关的尺寸。笔迹使用逻辑画布坐标，画布固定为 `1024×1624`。相机使用 `zoom + center`，视口变化调用 `withViewport`，保留逻辑中心与缩放倍率。

手势协程使用稳定 key，不因相机每帧变化而取消。单次 transform 手势维护局部最新相机。每帧先计算旧相机下实时质心对应的逻辑点，再计算新缩放，并令该逻辑点投影到 `centroid + pan`，然后钳制相机边界。

## 布局

竖屏绘图区域在工具栏和底部操作栏之间居中，以可用宽高共同约束 `0.6306075`。横屏使用固定工具栏与居中的比例画布，不再让 Canvas 任意拉伸填满剩余区域。学习页同样按可用宽高约束，并增加 `10%` 水平留白。

## 兼容性

旧版 `1024×2048` 图片继续以 FitCenter 显示，不裁切、不拉伸。新保存图片写为 `1024×1624`；数据库表结构不变，不做迁移。
