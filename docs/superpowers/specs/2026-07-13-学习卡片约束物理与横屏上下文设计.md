# 学习卡片约束物理与横屏上下文设计

## 手势物理

- 内圈半径固定为屏幕宽度 `1/5`，圈内最大倾斜 `8°`。
- 首次越圈时按位移主轴锁定为纵向反馈或横向翻面，一次手势内不再切换。
- 纵向模式越圈后固定为 `7°` 微倾，旋转轴锁定为越圈瞬间的轴；继续增大 `dy` 只改变中心位置。
- 正反面共享绝对旋转轴。背面纹理补偿使用与正面相反的轴符号，避免接近 `180°` 法线时出现平面内旋转。
- 横向模式在越圈后额外移动屏宽 `1/10` 完成 `180°` 翻面。

## 释放与离场

纵向释放使用：`projectedDy = dy + effectiveVelocityY × 0.14s`。末速度低于 `max(系统最小 fling, 600dp/s)` 时不贡献投影。越圈且投影位移达到屏高 `28%` 时触发反馈。

反馈成立后，卡片中心动画到屏幕外，角度保持松手值不变；完全离场后才提交反馈并切换下一张。全程不使用透明度或阴影。

## 横屏布局

- 仅卡片页保留永久上下文栏；卡片列表和详情各占导航栏后剩余宽度的一半。
- 学习页隐藏标题和状态行，不创建上下文栏，卡片浮于完整舞台。
- 录入页不创建上下文栏；`208dp` 紧凑工具面板无滚动，颜色下方使用横向笔刷 Slider。
- 备份页当前不创建上下文栏。
