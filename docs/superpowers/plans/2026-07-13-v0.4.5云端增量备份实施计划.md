# v0.4.5 云端增量备份实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在现有备份页内实现基于 AList WebDAV 的手动增量备份与版本恢复。

**Architecture:** 云端使用完整元数据快照和 SHA-256 图片对象；`index.json` 是最近 8 个版本的唯一入口。ViewModel 统一驱动本地 SAF 与云端操作，Compose 页面只渲染状态并发出命令。

**Tech Stack:** Kotlin、Jetpack Compose、kotlinx.serialization、OkHttp、WebDAV、DataStore/应用私有配置。
